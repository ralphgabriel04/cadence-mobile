# 📖 Glossaire

| Terme | Définition |
|-------|------------|
| **WACAP** | Weekly Active Coach-Athlete Pairs — North Star metric de Cadence |
| **RLS** | Row Level Security — Politique de sécurité PostgreSQL au niveau des lignes |
| **RPE** | Rate of Perceived Exertion — Échelle 1-10 d'effort perçu par l'athlète |
| **PR** | Personal Record — Record personnel sur un exercice (aussi : Pull Request en contexte dev) |
| **1RM** | One-Rep Max — Charge maximale pour une répétition |
| **Readiness** | Formulaire quotidien mesurant le niveau de préparation de l'athlète |
| **Session** | Une séance d'entraînement dans un programme |
| **Program** | Un plan d'entraînement structuré créé par un coach |
| **EAS** | Expo Application Services — Service de build et déploiement d'Expo |
| **OTA** | Over-The-Air update — Mise à jour JS sans passer par les stores |
| **Deep linking** | Lien qui ouvre directement un écran spécifique de l'app |
| **NativeWind** | Framework CSS-in-JS pour React Native basé sur Tailwind CSS |
| **Supabase** | Backend-as-a-Service open source (alternative à Firebase) |
| **RN** | React Native — Framework mobile cross-platform |
| **MVP** | Minimum Viable Product — Version minimale viable du produit |
| **NPS** | Net Promoter Score — Mesure de satisfaction et recommandation |
| **DoD** | Definition of Done — Checklist obligatoire avant de merger une PR |
