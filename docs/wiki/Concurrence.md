# 🏆 Concurrence

## Top 5 concurrents

### 1. Hevy
- **Forces** : UX exceptionnelle, communauté active, gratuit généreux
- **Faiblesses** : Pas de coaching intégré, focus self-training
- **Ce qu'on prend** : Qualité de l'UX workout logging, animations fluides

### 2. TrueCoach
- **Forces** : Leader du marché coaching, intégrations riches
- **Faiblesses** : UX datée, cher, athlète = afterthought
- **Ce qu'on prend** : Modèle de données coach-athlète, feature set de référence

### 3. Everfit
- **Forces** : All-in-one (workout + nutrition + habits), bonne app mobile
- **Faiblesses** : Complexe, overwhelming pour les nouveaux coachs
- **Ce qu'on prend** : Concept de readiness/check-in, multi-metric tracking

### 4. TrainHeroic
- **Forces** : Marketplace de programmes, communauté, force/conditioning focus
- **Faiblesses** : Design austère, courbe d'apprentissage élevée
- **Ce qu'on prend** : Concept de marketplace (post-MVP), analytics de performance

### 5. Trainerize
- **Forces** : Intégrations (Mindbody, Stripe), automatisation
- **Faiblesses** : UX pire du lot, bloated, lent
- **Ce qu'on prend** : Business model (ce qu'il faut facturer), intégrations à considérer

## Stratégie "Best of Market"

> Prendre le **meilleur** de chaque concurrent et l'exécuter avec une **UX supérieure**.

| Feature | Meilleur concurrent | Notre approche |
|---------|--------------------|----------------|
| Workout logging | Hevy | Même fluidité, mais avec coaching intégré |
| Coach-athlete relation | TrueCoach | Même profondeur, mais UX moderne |
| Readiness tracking | Everfit | Plus simple, plus visuel |
| Analytics | TrainHeroic | Mêmes insights, meilleur design |
| Pricing | Trainerize | Plus transparent, freemium généreux |

## Ce qui nous différencie

Aucun concurrent ne combine :
1. UX premium (comme Hevy)
2. Coaching intégré (comme TrueCoach)
3. Simplicité (anti-Trainerize)
4. Marché québécois (bilingue FR/EN)
5. Prix accessible (freemium)
